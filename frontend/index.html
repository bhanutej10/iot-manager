Click a button to generate a Matplotlib chart via backend API
How it works: Backend calls _mpl_timeseries() or _mpl_dashboard() using Matplotlib with a dark style context, renders to a BytesIO buffer, base64-encodes and returns via REST. The multi-panel dashboard uses GridSpec(2,3) with temperature trends, CO₂ fill, battery bars, alert pie, and RSSI heatmap panels.

Quick start commands
# Full stack (Docker required)
$ cd iot-simulator
$ docker compose up -d
$ open http://localhost
# Local (no Docker)
$ pip install -r backend/requirements.txt
$ python backend/app.py &
$ SENSOR_INTERVAL=3 bash scripts/simulate_sensors.sh &
$ open iot_dashboard.html
# MQTT subscriber (separate terminal)
$ python backend/mqtt_subscriber.py
